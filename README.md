# 🧩 My First Jenkins CI/CD Pipeline

This repository contains my first Jenkins CI/CD pipeline project created during my **DevOps Internship at Elevate Labs**.  
The goal of this project is to automate the build and deployment process using **Jenkins** and **GitHub**.

---

## 🚀 Project Overview

The main objective of this task was to:
- Install and configure Jenkins on Windows (without NSSM)
- Create a Jenkins pipeline that runs automatically
- Integrate Jenkins with GitHub
- Verify successful build execution

---

## ⚙️ Tools and Technologies Used

- **Jenkins** (Continuous Integration tool)  
- **Git & GitHub** (Version control and code hosting)  
- **Java (JDK 21)** (Required for Jenkins)  
- **Windows PowerShell** (For Git commands)
